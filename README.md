# react-scroll-spy

一个React版本的scroll-spy组件。


[Demo](https://xuelq007.github.io/react-scroll-spy/)


## 快速开始

```javascript

import ScrollSpy from './component/scrollSpy/scrollSpy'

<ScrollSpy layout='top' components={components} tabs={['Facebook', 'Google', 'Microsoft', 'IBM']} />

```
